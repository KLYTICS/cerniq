01 — Identity

Five marks. One recommended.

The Aegis Shield is the recommended primary — most distinctive, geometrically ownable, scales from favicon to billboard. The other four are kept on file as alternates for sub-brand surfaces, sticker packs, and motion treatments.

Primary
The Aegis Shield
Hexagonal stability. A in negative space. Audit-chain bisector. The one to ship.
The Halo
Most abstract — feels like infrastructure. Strong alternate for SDK and dev surfaces.
The North Star
Most editorial. "Switzerland of identity" made literal. Use on About / mission surfaces.
The Lattice
Hash chain made visual. Best for technical / SDK / docs surfaces. Sticker pack candidate.
The Verified Mark
Most direct semantically. Riskier (checkmarks crowded). Reserved as a verification badge motif.

Agent registration, public-key rotation, principal-scoped key management. The SDK is the only thing that ever sees a private key.

Hot path

Verify

The one endpoint relying parties code against. Edge-portable. Zero framework imports. Sub-50ms p99 budget.

Module

Audit

Append-only log. Every event signed and chained to the previous. UPDATE and DELETE never run.

Code

// Verify an agent's signed action — the AEGIS hot path
import { aegis } from "@aegis/sdk";

const verdict = await aegis.verify({
  agent: "agt_9f3a…c8d2",
  action: "transfer.execute",
  signature: req.headers["x-aegis-sig"],
  spend:  { amount: 240, currency: "USD" }
});

if (verdict.status === "VERIFIED") return proceed();
if (verdict.status === "SPEND_LIMIT_EXCEEDED") return deny(verdict);
